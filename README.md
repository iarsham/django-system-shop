### How to Run 🏃‍♀️

```shell
1 git clone https://github.com/iarsham/django-system-shop.git
2 cd django-system-shop
3 python -m venv venv
4 source venv/bin/activate
5 pip install -r requirements.txt 
6 python manage.py runserver

```
